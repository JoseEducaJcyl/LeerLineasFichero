# Contador de Archivo de Texto

## Descripción
Aplicación de consola en Java que lee un archivo de texto y muestra estadísticas básicas: número de líneas, palabras y caracteres.

## Funcionalidades
- Leer archivo "datos.txt"
- Mostrar el contenido del archivo
- Contar líneas totales
- Contar palabras (separadas por espacios)
- Contar caracteres totales

## Tecnologías
- Java
- BufferedReader
- FileReader
- Manejo de excepciones IOException
